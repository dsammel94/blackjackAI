To run learning algorithm, run Learn.py from a console.
To run the blackjack simulator with the learned strategies, run BlackJack.py from the console.
All .py files are described in the Final Report

.csv files:

- output.csv: The file containing the probability matrix for Learner
- highhit.csv: The file containing the probability matrix for when the count is high for CountingLearner
- lowhit.csv: The file containing the probability matrix for when the count is low for CountingLearner
- midhit.csv: The file containing the probability matrix for when the count is neutral for CountingLearner
- stats.csv: The file containing the rate of learning data for Learner
- stats2.csv: The file containing the rate of learning data for CountingLearner
